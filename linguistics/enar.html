The letter used for a glottal stop in Arabic, ا, is used as an empty letter for when a word starts with a vowel and you need to put a diacritic on the letter before to show it but there is no such letter.

The vowels are more complicated. Arabic script shows vowels with diacritics generally, and it only has marks for /a i u/, while English has 11 vowels in my own variety. To account for this, the pharyngeal consonant letters ح and ع are used for /e ɒ/ respectively.

We can write /a ɪ ʊ/ by placing the appropriate diacritic on the preceding consonant: َ◌ for /a/, ِ◌ِ for /ɪ/, ُ◌ for /ʊ/..

As for the schwa, this can be indicated with the diacritic ْ◌. This is used to indicate that a consonant is not followed by a vowel, but it would be weird to use it for this purpose in English (a word like con would be more complex than comma).

To indicate the vowels /i/ and /u/, we can use ي and و with the diacritic ِ◌ِ (for /i/) or ُ◌ for /u/. . The same goes for the diphthongs /ai au/.

For /ei ou oi/, it is more convenient to add the appropriate diacritic to ح and ع: حِ is /ei/, عُ is /ou/, and عِ is /oi/. For /ɛː ɔː/ we add the schwa diacritic: حْ is /ɛː/, عْ is /ɔː/. For /ɜː/, in dialects that distinguish it and /ɛː/, we can combine the two, front first: حع.

In those dialects that distinguish /ʌ/ we can use the diacritic for /a/ on the preceding consonant, followed by عِ.
