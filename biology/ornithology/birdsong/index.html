This page can be used to help you learn the songs of British birds. Press the Play button below, and a playlist of British bird songs (using sound files from xeno-canto) will begin playing, with the songs in a random order. The birds are all those of the order Passeriformes which have a page for them on the RSPB's website which links to a xeno-canto sound file. At some point I'll probably add some more birds as well as customization options.
